# Iris.Plugins.Contracts

`Iris.Plugins.Contracts` contains the public contracts required to build Iris plugins.

It includes:

- `ISource`
- `ITarget`
- `DataMessage`
- `PluginAttribute`
- `PluginType`
- `IPluginMetadata`

This package is intended to be referenced by external plugin projects without pulling in the full host implementation.
